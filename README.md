## 技术栈
- 语言：Python3 (Django)
- 数据库: MySql、 redis
- 任务队列(异步处理): celery
- 分布式文件存储: FastDFS
- web服务器配置: Nginx+ uwsgi
- 开发环境： PyCharm、Linux、vim
